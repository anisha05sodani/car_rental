# RENTAL CAR SYSTEM
A simple **Rental Car System** implemented in **Java** using **Object-Oriented Programming (OOP)** concepts.

The system allows users to browse available cars, rent cars, and return them.
## Features

✅ List available cars  
✅ Rent a car  
✅ Return a car  
✅ Customer Management  
✅ Car Management  
✅ Rental History
